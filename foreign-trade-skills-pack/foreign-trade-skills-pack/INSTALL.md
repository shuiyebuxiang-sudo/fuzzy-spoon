# Foreign Trade Skills Pack

This pack contains 8 Codex skills:

- `trade-docs-helper` - make PI, CI, PO, certificate of origin, quote sheets, and customs documents.
- `trade-message-reader` - read customer messages and explain what the customer wants.
- `trade-reply-writer` - draft simple professional English replies.
- `trade-shipment-checker` - check shipment, customs, and export document readiness.
- `trade-quote-pi-maker` - prepare quotes and PI plans from model lists and quantities.
- `trade-order-tracker` - track order steps from inquiry to shipment and after-sales.
- `trade-price-checker` - check prices, totals, currency, and discount issues.
- `trade-customer-developer` - develop leads, write outreach, and maintain follow-up notes.

## Install On Windows

1. Unzip this folder.
2. Copy the 4 skill folders into:

```text
C:\Users\<YourUserName>\.codex\skills
```

Or run:

```powershell
.\install-windows.ps1
```

## Install On macOS / Linux

Copy the 8 skill folders into:

```text
~/.codex/skills
```

## Use

After installation, restart Codex.

Example prompts:

```text
Help me understand this customer message.
Help me make a PI from this customer request.
Help me reply to this customer in English.
Help me check if these shipment documents are complete.
```
